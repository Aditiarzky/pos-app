- **Hybrid Routing & API**: Leverages Next.js App Router for both UI pages (`src/app/dashboard/*`) and backend API endpoints (`src/app/api/*`), co-locating business logic with its consumption.
- **Shared Infrastructure Wiring**: Centralizes cross-cutting concerns in `src/lib/`, where `db.ts` provides a cached, environment-aware Drizzle ORM instance (supporting both local PostgreSQL and Cloudflare Hyperdrive), and `auth.ts` manages JWT session lifecycle via httpOnly cookies.
- **Client-State Integration**: Uses a root `QueryProvider` to inject TanStack Query across the client component tree, enabling standardized data fetching hooks (`src/hooks/*`) that consume the internal `/api` routes via a configured Axios instance with automatic token refresh interceptors.
- **Schema & Validation Boundary**: Enforces data integrity through a single source of truth in `src/drizzle/schema.ts`, extended by Zod validators in `src/lib/validations/` used by both API route handlers and client-side forms.
